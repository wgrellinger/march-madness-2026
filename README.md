# march-madness-2026

Used various metrics and statistics as well as XGB and Random Forest to predict the 2026 March Madness bracket. Also created variables using the Bradley Terry model and creating a Strength of Schedule variable. Used a bit of Claude to make it.

All files were made using March Machine Learning Mania 2026 data on kaggle linked [here](https://www.kaggle.com/competitions/march-machine-learning-mania-2026/discussion?sort=hotness).
